## Website 2: https://developer.mozilla.org

### 1. Find the navigation menu – what tag is it wrapped in?
The navigation menu is wrapped in a **`<nav>`** tag.

### 2. How is the search bar structured?
The search bar is usually built using:
- a wrapping container
- a **`<form>`** element
- an **`<input>`** field for typing the search text
- sometimes a **button** or icon for submitting the search

### 3. What happens when you hover over links (check the styles)?
When hovering over links:
- the text color may change
- the link may become underlined
- the cursor changes to indicate it is clickable
- hover styles are applied through CSS

## Website 3: GitHub (https://github.com)

### 1. Identify 5 different HTML elements
Five HTML elements found on GitHub include:
- `<header>`
- `<nav>`
- `<main>`
- `<form>`
- `<input>`

### 2. Find a form element and list its inputs
One form element on GitHub is the search form.

Inputs include:
- a search **`<input>`** field
- hidden inputs used for search handling
- a submit button in some cases
